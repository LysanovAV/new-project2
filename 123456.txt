Работа с удаленным репозиторием
git push -u origin master - отправка в удаленный репазеторий + установка связи, что бы в бальнейшем отправлять без параметров
git push - отправка в удаленный репазеторий

git remote -v - просмотр пути к удаленному репозеторию
git remote remove origin - уделение пути к сущ. репозетрию
git remote add oridgin git@github.com:LysanovAV/new-projekt4.git(путь) - прописывание пути к удаленному репозеторию (origin - название его)
git remote set-url origin “адрес репозитория на новом хостинге” - сменить веб-хостинг для вашего локального репозитория

git pull - заятнуть изменения с сервера
:wq - команда сохранения сообщения коммита, при конфликте версий
